Mooch

Created at Daemon Dash 2015 (September 26, 2015 - September 27, 2015)

Source Code

Mooch is a Chrome extension that allows the user to share some accounts with their friends, family, and trusted persons without transferring any login information (i.e. emails and passwords). The extension communicates with a backend server written in Django and backed by PostgreSQL via a RESTful API.

ArtRoom

Created at YHack 2014 (October 31, 2014 - November 2, 2014)

Source Code

ArtRoom is an Android application that allows users to, in an art museum context or such, scan an art object and promptly get information regarding the scanned object. It is intended to generate interest in art and art history by integrating technology and making it simple to learn about art. Additionally, users can discuss and comment on the scanned object with others via Moxtra.

Awards

  • Runner-up for Best Use of Google APIs at YHack 2014

Friendcast

Created at HackRU Fall 2014 (October 11, 2014 - October 12, 2014)

Source Code

Friendcast is an Android application that is meant to make it simpler to get together with your friends. This can be extended to be used for multiple purposes from eating out to having fun. The app scans the user's location and, using the Google Maps API, finds nearby food locations. Once a location is chosen, the user can then select which of their Facebook friends (who also have Friendcast installed) to notify; they can then choose to accept or reject the invitation to go eat.

Awards

  • Favorite Yodle Hack at HackRU Fall 2014

CSGO Bets

Released: August 2014

Source Code

CSGO Bets was my first and only Android app published on the Google Play Store. Its main purpose was to easily allow users to view information regarding matches/bets from csgolounge.com. Web scraping (jsoup) was utilized to get matches and compile match information.

Whatchamacallit

Created at HackRU Spring 2014 (April 12, 2014 - April 13, 2014)

Source Code

Whatchamacallit is a Google Glass app that allows the user to say a short phrase or rough definition of a word that they are thinking of, but can't quite grasp. Upon saying their phrase/definition, the app will attempt to find and display the most closely related words.
